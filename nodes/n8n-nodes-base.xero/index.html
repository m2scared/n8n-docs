# Example Usage

This workflow allows you to get upto 100 invoices from Xero. You can also find the workflow (opens new window) on the website. This example usage workflow would use the following two nodes.

  • Start
  • Xero

The final workflow should look like the following image.

# 1. Start node

The start node exists by default when you create a new workflow.

# 2. Xero node

  1. First of all, you'll have to enter credentials for the Xero node. You can find out how to do that here.
  2. Select 'Get All' from the Operation dropdown list.
  3. Select the organization for which you want to get the invoices from the Organization ID dropdown list.
  4. Click on Execute Node to run the workflow.
